%%  When filling in this template, the LLM should:
%%
%%  1. Replace all {{PLACEHOLDER}} tokens with real content from the profile.
%%  2. Remove optional sections (Projects, Certifications, Languages) if data
%%     is not available — do not leave placeholder tokens in the output.
%%  3. Write achievement bullets in the STAR format (Situation, Task, Action,
%%     Result) using strong action verbs: Led, Built, Reduced, Improved, etc.
%%  4. Quantify achievements wherever possible: "Reduced load time by 40%".
%%  5. Mirror keywords from target job descriptions for ATS matching.
%%  6. Keep the Professional Summary to 3-4 sentences max.
%%  7. Do NOT include: photos, tables, text boxes, colored text, or decorative
%%     symbols — all of these can break ATS parsing.
%%  8. Dates must follow the format: "Jan 2022 -- Mar 2024" or "2022 -- 2024".
%%  9. {{PORTFOLIO_LINE}} should be:
%%       $\,|\,$ \href{URL}{Portfolio}
%%     or empty string "" if no portfolio URL is provided.
